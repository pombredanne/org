Introduction

This document lays out the feature and API set for the next version of the MWE (version 1.1.0). This project plan inherits from the Modeling Project Plan, which should be referenced when consulting this individual project plan.

Release Deliverables

The release deliverables have the same form as is found in most Eclipse projects, namely:

  • EMFT MWE SDK (includes runtime and tooling components, with sources, examples, and documentation) (downloadable and update site).
  • EMFT MWE runtime binary distribution (downloadable and update site).
  • EMFT MWE tests (downloadable and update site)
  • MWE2 SDK (The new version. Includes runtime and tooling components, with sources, examples, and documentation) (downloadable and update site).
  • MWE2 runtime binary distribution (downloadable and update site).
  • MWE2 tests (downloadable and update site)

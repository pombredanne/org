Project description

The Web Tooling Platform (WTP) Project is made up of two subprojects, Web Standard Tools (WST) and J2EE Standard Tools (JST). In addition to depending on the Tools Platform, WST depends on EMF, XSD InfoSet, and GEF, while JST depends on WST and JDT.

Refer to the WTP Project overview for additional WTP Project information. You can click on the image below for additional information for the JST and WST sub-projects.

Target Environments

The server-side part of the CloudFree Commerce Platform will be written and compiled against:
  • Equinox Version 3.5
  • Version 6.0 of the Java Platform APIs (i.e., Java 2 Platform, Release 6.0 SE)
  • Java Servlet API Version 2.5

Internationalization

The CloudFree Platform is designed as the basis for internationalized products. The user interface elements provided by the CloudFree components, including dialogs and error messages, are externalized. The English strings are provided as the default resource bundles.

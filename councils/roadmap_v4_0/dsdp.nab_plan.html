Introduction

The Native Application Builder project creates framework to build a C++ and several languages GUI applications. The goal is to allow the user to write a GUI application once and build it for many platforms without changing the code. Conceptually, this is similar to SWT and JFace, with the notable exception that the NAB tool is written in C++ and designed for use in C++ applications.

    The Eclipse Platform and SuSE Linux

    The Synergies - why is it import
  • Linux provides a means of consolidating hardware platforms through one common OS
  • SuSE Linux provides a Linux version for all relevant hardware platforms, .e.g. all four IBM eServer platforms, Sparc
  • Eclipse based tools will be available on these platforms

    The Benefits
  • Easy access for programmers
  • One working environment for all hardware architectures

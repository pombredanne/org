    The Eclipse Platform and SuSE Linux

    The Synergies
  • Eclipse will conquer hardware and OS platforms.Eclipse supports Windows, Unix and Linux
  • Eclipse is available for SuSE Linux – now!.The Eclipse Platform gets built automatically on all hardware architectures supported by SuSE Linux. The Intel Architecture version is already released, others will follow. The rpms will continue to show up for download on eclipse.org – up to date and well maintained

    The Eclipse platform will
  • Allow programmers to run projects on their native setups
  • Allow programmers to migrate their projects smoothly to Linux
  • Substantially ease porting applications to hardware architectures running SuSE Linux .
    and through all this increase the amount of applications available on Linux dramatically ?

What does Fujitsu contribute to Eclipse?
Fujitsu will be able to develop and provide tools for Eclipse as
follows:
COBOL development tool
Embedded Java application development tool(for mobile phone,
PDA … etc.)
Any tools related to Java
Fujitsu will be able to contribute Eclipse customization for
internationalization.
Fujitsu will be able to contribute to improving the quality of Eclipse
through the quality assurance process for our Eclipse-based
products.
    Fujitsu develops products using Eclipse. The results of the testing of our
products will be shared with the community.
 Fujitsu will be able to contribute to evangelizing Eclipse in Japan
because of Fujitsu’s development tool customer base.

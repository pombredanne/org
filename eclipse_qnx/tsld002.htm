Eclipse Slide

    Eclipse: Breaking the Proprietary Locks
    One standard development environment:
  • For all projects
    - developers can plug in tools they need
    - just one user interface to learn
  • For all tools vendors
    - no need to develop multiple versions of the same tools
    - lower development and support costs
  • For any supported OS
    - developers can choose the best OS for the task at hand

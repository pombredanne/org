BEA and Sybase Join Eclipse Foundation as Strategic Developer Members

February 22, 2005 -BEA and Sybase announced that they are becoming Strategic Developer Members of the Eclipse Foundation and will be joining the Eclipse Board of Directors. In addition, it was announced that Sybase is proposing a Data Tools Project. BEA announced that it is proposing a Language Development Toolkit project and will be contributing to the Web Tools Platform project. Tim Wagner of BEA has joined the WTP PMC and has been elected as co-leader of the project by the PMC.

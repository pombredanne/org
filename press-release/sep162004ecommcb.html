Eclipse Community Bulletin

Project Proposal for
Eclipse Communications Project

The Eclipse Foundation has announced a proposal for a new Technology Project called Eclipse Communications Project. The goal of the Eclipse Communications Project is to provide an open source, extensible, peer-to-peer communications framework for the Eclipse platform. A proposal for this project is now available for the next 30 days for review and feedback.

The Eclipse Communications Project (ecomm) will focus on improving the facilities within Eclipse for building secure, reliable, peer-to-peer messaging and communications applications. Application developers will be able to use ecomm APIs/plugins to easily implement applications that require file sharing, instant messaging, data/voice/video conferencing, shared modelling and editing as well as other integrated real-time communications facilities.

To review the proposal and provide feedback please visit the eComm proposal home page on www.eclipse.org/proposals/ecomm/index.html.
